# ComsPlus

ComsPlus adds a local privacy display name and an in-level chat for players who use the mod together with Globed.

## Privacy

The fake name is local UI text. It does not change your Geometry Dash account, account id, RobTop login, or Globed identity. When enabled, ComsPlus replaces visible labels that contain your own username so your real name is less likely to show on stream.

## Chat

The chat opens from a draggable floating bubble in levels. Opening it pauses the level, and pressing Resume returns it to the bubble. Messages are sent through Globed 2.2.0+ server events to other ComsPlus users connected to the same Globed level.
